# Vital Core App

A Flutter application for the Stitch Mawisa Medical Practice.

## Getting Started

This is a Flutter project. To get started and run the application locally, follow these steps:

### Prerequisites
1. Ensure you have [Flutter installed](https://docs.flutter.dev/get-started/install) on your machine.
2. Verify your installation by running:
   ```bash
   flutter doctor
   ```

### Running the App
1. Open a terminal and navigate to the project directory:
   ```bash
   cd vital_core_app
   ```
2. Get the required dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app on your connected device or emulator:
   ```bash
   flutter run
   ```

To run specifically on the web (if Chrome is installed):
```bash
flutter run -d chrome
```
